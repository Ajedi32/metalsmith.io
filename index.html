How does it work?

Metalsmith works in three simple steps:

  1. Read all the files in a source directory.
  2. Invoke a series of middleware that can manipulate the files as they please.
  3. Write the results to a destination directory.

The middleware plugins are invoked with the contents of each file, and any optional YAML front-matter.

What makes Metalsmith powerful is that the plugins can manipulate the input any way they want. They could convert Markdown files to HTML, compress all images files, omit files by filename, add a generated RSS feed file, anything!

Plugins

Metalsmith core doesn't have any plugins bundled with it by default, you just require plugins as needed. If you make your own plugin, submit a pull request and it will show up here.

  • Permalinks

    Rename files so that they are permalinked properly for static sites.

  • Templates

    Render any file through a template engine.
